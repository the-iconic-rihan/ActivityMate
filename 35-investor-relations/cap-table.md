# Cap Table

**Status:** Draft v1 — illustrative structure only.

## Scope

Ownership structure, with a confidential-handling note.

## Confidentiality note

**This document, once populated with real figures, is highly confidential.** The cap table must never be committed to a public or shared repository in its real form. This file should carry the illustrative *structure* only until the founders replace it with actual figures in a restricted-access location (a captable tool such as Carta/Pulley, or a locked spreadsheet shared only with counsel, the board, and named investors under NDA). If this file is ever populated with real ownership percentages, move it out of any repo location with broader read access before doing so, and note that relocation in [data-room-checklist.md](data-room-checklist.md).

## 1. Illustrative pre-seed cap table

*All figures below are illustrative placeholders showing a plausible structure for an Indian pre-seed consumer startup — not ActivityMate's real ownership. Founders must replace every number before this document is used for any actual investor conversation.*

| Holder | Instrument | Illustrative % (fully diluted) |
|---|---|---|
| Founder 1 (CEO) | Common / founder equity | 42% |
| Founder 2 (CTO) | Common / founder equity | 38% |
| ESOP pool (unallocated, reserved) | Option pool | 10% |
| Pre-seed investor(s) | SAFE / CCPS, converting at next priced round | 10% |
| **Total** | | **100%** |

Notes on the illustrative structure:
- A 10% unallocated ESOP pool reserved pre-close is standard practice so the pool dilutes founders and pre-seed investors proportionally rather than only post-round hires. The actual pool size should be re-checked against the real hiring plan in [hiring-plan-org-chart.md](../36-hiring-people-culture/hiring-plan-org-chart.md) — if 15+ hires are planned in year one, 10% may be tight.
- Two co-founders is the assumed structure here for illustration; if ActivityMate has a different founder count or split, replace this table's rows accordingly — do not force real data into a two-founder shape.
- Instrument choice (SAFE vs. CCPS/convertible note) should be set by counsel based on the actual investor and the applicable Indian company structure (most likely a private limited company under the Companies Act, 2013, possibly with a flip to a foreign holding entity if raising from foreign VCs — a decision for counsel, not this document).

## 2. Illustrative seed-round cap table (post pre-seed conversion)

| Holder | Illustrative % (fully diluted) |
|---|---|
| Founders (combined) | 62% |
| ESOP pool (expanded) | 13% |
| Pre-seed investors (converted) | 10% |
| Seed-round investors (new) | 15% |

This second table exists to show the *shape* of dilution across a round, not to predict actual seed terms. Real seed dilution depends on valuation, round size (see [financial-model.md](financial-model.md) Section 7 for the illustrative funding requirement), and negotiated terms.

## 3. What this document does not cover

- Actual valuation (pre-money/post-money) — set per round by founders and counsel based on market comparables at the time of raise.
- Vesting schedules for founder and ESOP equity — should be documented in [compensation-equity-framework.md](../36-hiring-people-culture/compensation-equity-framework.md), not duplicated here.
- Legal instrument terms (liquidation preference, pro-rata rights, board seats) — owned by term sheets and definitive agreements, not this summary document.

## 4. Maintenance

This document should be updated at every priced round or SAFE/note conversion event, and reviewed alongside every board deck per [board-deck-template.md](board-deck-template.md). Ownership of keeping it current sits with the CEO/founders, per this folder's ownership in [README.md](README.md).
